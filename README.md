# Database-Webapp COMP1004

A simple web application built with **HTML, CSS, and JavaScript** that connects to a **Supabase PostgreSQL database**.  
This project was completed as part of the **COMP1004 Web Programming coursework**.

The application allows users to:

- Search for **people** by name or licence number
- Search for **vehicles** by registration number
- **Add new vehicles**, including selecting an existing owner or creating a new one

Supabase is used as the backend, accessed through the Supabase JavaScript client.
